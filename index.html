Colorado Resource Viewer


By Jamie Braun


The Colorado Resource Viewer Application is an application intended for scientists, analysts, surveyors, land managers, etc. The primary purpose of this application is to examine the various datasets to get an understanding of the current resource extraction landscape of Colorado.


The right panel of the application contains the widgets that accompany each layer. They will open and close when layers are selected in the layer selector to the left of the widget panel. Each widget summarizes some of the basic information to be learned about each resource type.


Each layer can be turned on and off using the layer selector, and if a polygon or point feature is clicked, a popup will display relevant information.
